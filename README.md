# Job Market ETL Pipeline

An end-to-end data engineering project to extract, transform, and analyze Analytics and Data Engineering job postings using Python, dbt, DuckDB, and Kubernetes.

## Tech Stack
- **Ingestion:** Python (Requests)
- **Storage:** DuckDB
- **Transformation:** dbt (data build tool)
- **Orchestration:** Kubernetes
- **API:** RapidAPI (LinkedIn/Job Search)
